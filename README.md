# BartCraft Transmog Addon

This addon is required to use the BartCraft transmog system in-game.

## Installation

Place the full **Transmog** folder here:

```text
World of Warcraft/Interface/AddOns/Transmog
```

Then restart your game client and make sure **Transmog** is enabled on the AddOns screen.

## BartCraft Launcher Users

No manual download is needed if you use the **BartCraft Launcher**.

Just open the launcher, click **Update**, and the Transmog addon will be installed automatically.

## Where to Transmog

Visit **Magister Stellaria** in:

* Orgrimmar — Valley of Spirits
* Stormwind — Mage Tower
* Shattrath — BartHub

Customize your look and start mogging!
